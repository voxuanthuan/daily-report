# Go-TUI Promotional Video

This project was built with [Remotion](https://www.remotion.dev/).

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

## Preview

Start the local preview server:
```bash
npm start
```

## Render

Render the video to an MP4 file:
```bash
npm run build
```

The output file will be in the `out` directory.

## Assets
AI-generated assets are located in the `public` directory.
