# Jira Daily Report - Promotional Video

12-second promotional video for Jira Daily Report TUI tool.

## Setup

```bash
npm install
```

## Preview

```bash
npm start
```

## Render

```bash
npm run build
```

Output: `out/promo.mp4`

## Scenes

- **Scene 1** (0-3s): Frustrated developer scrolling Jira  
- **Scene 2** (3-7s): Terminal command typing and execution
- **Scene 3** (7-12s): Happy ending with CTA

## Assets

All visual assets are in `public/`:
- `frustrated_developer_*.png`
- `happy_coworker_*.png`
- `minimal_logo_*.png`
- `uploaded_image_*.png` (TUI screenshots)
